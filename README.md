## Установка в Chrome, Edge:
### 1. Качаем архив проекта
![Загрузка](https://user-images.githubusercontent.com/94071543/227984726-b9ede221-bca2-4d28-b8e8-8a1024eeab74.png)
### 2. Извлекаем в место откуда случайно не удалим(при удалении папки расширения, оно пропадёт в браузере)
![Извлечение](https://user-images.githubusercontent.com/94071543/227985153-e3cf55e2-4c46-4ae3-b855-ede74595c6d6.png)
### 3. Запускаем браузер &#9658; Открываем меню "Расширения" &#9658; Включаем режим разработчика &#9658; Нажимаем "Загрузить распакованное расширение" и выбираем папку с ресурсами расширения или перетаскиваем папку расширении в окно браузера
![Установка](https://user-images.githubusercontent.com/94071543/227989482-293c6471-35b7-4a5f-b2e0-812f98826eb4.png)
